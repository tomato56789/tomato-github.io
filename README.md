# 複製生物介紹網站

## 專案介紹

本專案是一個使用 **HTML + CSS + JavaScript + jQuery** 製作的教學型網站，主題為「生物複製技術」。

網站內容介紹了複製技術的基本概念，並以幾個代表性的案例說明：

* 桃莉羊（第一隻體細胞複製動物）
* 黑足鼬（瀕危動物複製）
* 靈長類複製（中中與華華）

本專案目標是練習前端開發技術，並結合科學知識做出具互動性的網頁。

---

## 使用技術

* HTML5（網頁結構）
* CSS3（版面與樣式設計）
* JavaScript（基本互動）
* jQuery（動畫與互動效果）
* VSCode（開發工具）

---

## 功能特色

### 基本功能

* 生物複製技術介紹
* 圖片展示（桃莉羊、黑足鼬、靈長類）
* 外部連結（連到 Wikipedia）

### 互動效果

* 點擊按鈕顯示 / 隱藏內容
* 淡入淡出動畫
* 滑動動畫
* 滑鼠懸停變色效果
* 點擊圖片放大效果
* 滾動時區塊淡入動畫
* 回到頂部按鈕
* 側邊隱藏式導航欄（漢堡選單）
